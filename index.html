Capabilities

What AutoSweep can do.

Every subsystem designed and built by our team — hardware, firmware, and software working as one.

F — 01
Real-time SLAM mapping

Builds and updates a complete floor plan of unknown rooms while navigating, using GMapping with wheel-odometry fusion.

GMapping · ROS2
F — 02
Multi-sensor fusion

LiDAR, ultrasonic, and IR sensors are merged through a Kalman filter for robust, low-noise obstacle detection at 10 Hz.

Kalman filter · STM32
F — 03
Adaptive path planning

A* search over a boustrophedon-decomposed grid finds the shortest coverage path and replans when obstacles appear.

A* · Coverage planning
F — 04
Surface-aware ML

A TFLite classifier running on-device detects floor type — carpet, tile, hardwood — and adjusts suction and speed in real time.

TFLite · Edge ML
F — 05
Wi-Fi telemetry

Live sensor readings, battery state, and map progress stream to a web dashboard over MQTT for real-time remote monitoring.

MQTT · ESP32
F — 06
Auto-dock & recharge

When battery drops below threshold, the robot navigates autonomously back to its docking station using stored map data.

PID control · IR beacon

System architecture

From photon to clean floor.

Four tightly integrated processing layers — each one feeding the next — produce reliable autonomous behaviour.

01
Perception

LiDAR and ultrasonic sensors sweep at 10 Hz. A Kalman filter merges the streams into a clean, noise-reduced point cloud accurate to ±2 cm.

STM32 · 10 Hz fusion loop
02
Localization

GMapping SLAM processes the point cloud alongside wheel-odometry to maintain a continuous pose estimate on a growing occupancy grid.

ROS2 · GMapping SLAM
03
Planning

Free space is decomposed into cells. A* computes energy-efficient coverage paths and replans dynamically when new obstacles are detected.

A* · Boustrophedon decomposition
04
Actuation

Velocity commands drive dual DC motors via PID controllers. The on-device TFLite model classifies surface type and adjusts suction power in real time.

PID control · TFLite edge inference
